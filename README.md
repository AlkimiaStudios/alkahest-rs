# Alkahest.rs

## [[UNDER ACTIVE DEVELOPMENT]]

Alkahest is an open-source game engine created using Rust and OpenGL.

### Building

```shell
git clone https://github.com/AlkimiaStudios/alkahest-rs.git
cd alkahest-rs
cargo build
```

### Running

```shell
cargo run
```
